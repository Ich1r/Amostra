# AMOSTRA CULTURAL - Antígua e Barbuda

Um novo projeto em Flutter desenvolvido como parte de um trabalho acadêmico.

## Alunos
- Guilherme Nunes dos Santos
- Filipe Ichiro Rabelo Okoizume

## Tema do aplicativo
O aplicativo tem como tema **Antígua e Barbuda**, um país caribenho composto por duas ilhas principais. O app tem como objetivo apresentar informações culturais, históricas, turísticas e sociais do país, funcionando como um guia informativo interativo.
